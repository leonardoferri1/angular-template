Text Area

The <text-area> component is a custom textarea that implements ControlValueAccessor, working with ngModel and Reactive Forms.

Example

Code

<text-area placeholder="Enter your message" [(ngModel)]="message" [maxLength]="200" [rows]="5" [cols]="40" [invalidValidator]="isInvalid" (valueChange)="onValueChange($event)"> </text-area>

Properties

Property Type Default Description
placeholder string '' The placeholder text displayed inside the textarea.
maxLength number - Maximum number of characters allowed.
isReadonly boolean false Determines if the textarea is read-only.
disabled boolean false Disables the textarea.
rows number 3 Number of visible rows in the textarea.
cols number 30 Number of visible columns in the textarea.
invalidValidator boolean false Determines if the textarea should show invalid styling.

Events

Event Type Description
valueChange string | null Emitted when the textarea value changes.
