Date Picker

The <app-date-picker> component is a custom calendar date selector with popup functionality. It supports two-way binding with [(value)], manual selection, and quick access to todayâ€™s date.

Example

Selected: {{ selectedDate | date : "fullDate" }}

<app-date-picker [(value)]="selectedDate" theme="dark" (valueChange)="onDateChange($event)"> </app-date-picker> <p>Selected: {{ selectedDate | date : "fullDate" }}</p>

Properties

Property Type Default Description
value Date | null null The currently selected date. Supports two-way binding with [(value)].
theme string '' Theme class applied to the display and buttons (e.g., "dark", "light").

Events

Event Type Description
valueChange Date Emitted whenever the user selects a date or clicks "Today".
