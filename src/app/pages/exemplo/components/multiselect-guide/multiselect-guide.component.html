MultiSelect

The <multiselect> component allows users to select multiple options from a list. It implements ControlValueAccessor, making it compatible with both ngModel and Reactive Forms.

Example

Selecionados:

{{ selectedFruits | json }}
<multiselect [options]="cities" labelField="name" valueField="code" placeholder="Select cities" [(ngModel)]="selectedCities" (valueChange)="onCitiesChange($event)"></multiselect>

Properties

Property Type Default Description
options any[] [] The list of options to display in the multiselect.
placeholder string 'Select options' Text shown when no options are selected.
labelField string '' The property name used for the option labels.
valueField string '' The property name used for the option values.
isReadonly boolean false If true, the multiselect cannot be opened or modified.
multiselectDirection 'up' | 'down' 'down' Controls whether the dropdown opens above or below the input.
invalidValidator boolean false Marks the multiselect as invalid for validation purposes.
semResultado string 'No results found.' Message displayed when no options match the search query.
selectedValues any[] [] Currently selected option objects.

Events

Event Type Description
valueChange any[] Emitted when the selected options change.
