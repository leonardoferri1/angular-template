Currency Input

The <currency-input> component is a custom input field integrated with ng2-currency-mask. It implements ControlValueAccessor, allowing it to work with ngModel and Reactive Forms.

Example

<currency-input [(ngModel)]="amount" [prefix]="'$ '" [allowNegativeValue]="false" placeholder="Enter an amount" (valueChange)="onValueChange($event)"> </currency-input>
