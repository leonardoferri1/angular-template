Input de Texto

O componente <text-input> é um input customizado que implementa ControlValueAccessor, funcionando com ngModel e Reactive Forms.

Exemplo

Código

<text-input placeholder="Digite seu nome" [(ngModel)]="name" [maxLength]="20" [invalidValidator]="isInvalid" (valueChange)="onValueChange($event)" (onBlurEmit)="onBlur()"> </text-input>

Propriedades

Propriedade Tipo Padrão Descrição
placeholder string '' Texto de placeholder exibido no input.
maxLength number - Limite máximo de caracteres permitidos.
isReadonly boolean false Define se o campo é somente leitura.
mask string '' Máscara do campo (usando ngx-mask).
disabled boolean false Desabilita o campo.
type 'text' | 'password' | 'number' 'text' Tipo de input. Se password, exibe botão para mostrar/ocultar senha.
invalidValidator boolean false Define se o input deve ter estilo de inválido.
allowSpecial boolean false Impede caracteres especiais caso seja false.

Eventos

Evento Tipo Descrição
valueChange string | null Emitido quando o valor do input é alterado.
onBlurEmit string | null Emitido quando o input perde o foco.
