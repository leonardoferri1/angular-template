Text Input

The <text-input> component is a custom input that implements ControlValueAccessor, working with ngModel and Reactive Forms.

Example

Code

<text-input placeholder="Enter your name" [(ngModel)]="name" [maxLength]="20" [invalidValidator]="isInvalid" (valueChange)="onValueChange($event)" (onBlurEmit)="onBlur()"> </text-input>

Properties

Property Type Default Description
placeholder string '' The placeholder text displayed inside the input.
maxLength number - Maximum number of characters allowed.
isReadonly boolean false Determines if the input is read-only.
mask string '' Input mask (using ngx-mask).
disabled boolean false Disables the input field.
type 'text' | 'password' | 'number' 'text' Input type. If password, shows a button to toggle visibility.
invalidValidator boolean false Determines if the input should show invalid styling.
allowSpecial boolean false Prevents special characters if set to false.

Events

Event Type Description
valueChange string | null Emitted when the input value changes.
onBlurEmit string | null Emitted when the input loses focus.
