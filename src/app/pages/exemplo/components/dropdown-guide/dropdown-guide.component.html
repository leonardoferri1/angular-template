Dropdown

The <dropdown> component is a custom dropdown field with search capability. It implements ControlValueAccessor, making it compatible with both ngModel and Reactive Forms.

Example

<dropdown [options]="countries" labelField="name" valueField="id" placeholder="Select a country" [(ngModel)]="selectedCountry" (valueChange)="onCountryChange($event)"></dropdown>

Properties

Property Type Default Description
options any[] [] The list of options to display in the dropdown.
placeholder string '' Placeholder text shown when no option is selected.
labelField string '' The property name in the option object used for display.
valueField string '' The property name in the option object used as the value.
isReadonly boolean false If true, the dropdown cannot be opened or modified.
dropdownDirection 'up' | 'down' 'down' Controls whether the dropdown opens above or below the input.
invalidValidator boolean false Marks the dropdown as invalid for validation purposes.
noResult string 'No results found.' Message displayed when no options match the search query.
selectedValue any null Currently selected option object.

Events

Event Type Description
valueChange any Emitted when the selected option changes.
onKeyUp string Emitted when the user types in the search box.
