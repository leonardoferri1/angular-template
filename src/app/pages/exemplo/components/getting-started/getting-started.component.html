Welcome to my design system, a collection of components, utilities, and visual patterns created in Angular to speed up application development with consistency, accessibility, and best practices.

Reusable components that are decoupled.

Consistent design, inspired by modern UI/UX principles.

Easy integration with Angular projects (standalone or module-based).

Extensibility for customizing themes, styles, and behaviors.

Quality: built with Angular 18, RxJS, input masks, and best practices for NPM publishing.

Use the angular-design-system whenever you need to:

- Build Angular applications with a consistent visual style.

- Reuse tested and accessible components.

- Ensure different teams maintain the same UI standards.

- Have a ready-to-use UI boilerplate for new projects.

- Use the public library to create your own components.

Installation

npm install @leonardoferri/angular-design-system

Other third-party libraries used in building your library may also be required in the consumer project, such as:

npm install ngx-mask ng2-currency-mask
