Angular Design System — Leonardo Ferri

Bem-vindo ao meu design system, um conjunto de componentes, utilitários e padrões visuais criados em Angular para acelerar o desenvolvimento de aplicações com consistência, acessibilidade e boas práticas.

Componentes reutilizáveis e desacoplados.

Design consistente, inspirado em princípios de UI/UX modernos.

Integração simples com projetos Angular (standalone ou módulos).

Extensibilidade para customização de temas, estilos e comportamentos.

Qualidade: uso de Angular 18, RxJS, máscaras de input e boas práticas de publicação em NPM.

Use o angular-design-system sempre que precisar:

- Criar aplicações Angular com visual consistente.

- Reutilizar componentes testados e acessíveis.

- Garantir que equipes diferentes mantenham o mesmo padrão de UI.

- Ter um boilerplate de UI pronto para novos projetos.

- Utilizar a biblioteca pública para criar os próprios componentes.

Instalação

npm install @leonardoferri/angular-design-system

Outras bibliotecas de terceiros que forem utilizadas na construção da sua library podem ser necessárias no projeto consumidor, como:

npm install ngx-mask ng2-currency-mask
