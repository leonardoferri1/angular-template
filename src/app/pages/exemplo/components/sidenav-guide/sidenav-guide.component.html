Sidenav

The <app-sidenav> component provides a customizable side navigation panel with overlay support. It supports two-way binding on its opened state and emits events when closed.

Example

Menu Content

  • Dashboard
  • Settings
  • Logout
<app-sidenav [(opened)]="isOpened" (onNavClose)="handleClose($event)"> <div class="p-4"> <h3>Menu Content</h3> <ul> <li><a href="#">Dashboard</a></li> <li><a href="#">Settings</a></li> <li><a href="#">Logout</a></li> </ul> </div> </app-sidenav> <button (click)="isOpened = true" class="default-primary-button"> Open Menu </button>

Properties

Property Type Default Description
opened boolean false Controls whether the sidenav is open. Supports [(opened)] two-way binding.
tooltip string 'Fechar' Tooltip text displayed on the close button (useful for accessibility).

Events

Event Type Description
openedChange boolean Emitted whenever the sidenav's opened state changes.
onNavClose Event Emitted when the sidenav is closed via overlay click or close button.
